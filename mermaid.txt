# Executions

This section of the guide aims at showing the graphs of the scripts executions as provided by the system. 

## FDS

```mermaid
graph TD
    Start([Start Script]) --> CheckFile{Is .fds file present?}
    CheckFile -->|No| ExitError([Exit with Error])
    CheckFile -->|Yes| MeshDecomp{Is there only 1 Mesh<br/>& multiple CPUs?}
    MeshDecomp -->|Yes| RunPython[Run FDS Mesh Decomposition]
    MeshDecomp -->|No| RestartLogic
    RunPython --> RestartLogic[Update .fds file with<br/>new Mesh coordinates]
    
    
    %% Restart Management
    RestartLogic{Does .restart file exist<br/>& Restart not in MISC?}
    RestartLogic -->|Yes| InjectRestart[Inject RESTART=TRUE<br/>into &MISC/&TAIL]
    RestartLogic -->|No| Cleanup[Clean possible old simulation files]
    
    InjectRestart --> CoreCalc
    Cleanup --> CoreCalc
    
    %% Core & Thread Calculation
    CoreCalc[Calculate MPI Processes<br/>& OMP Threads<br/> from input FDS file]
    CoreCalc --> HardwareCheck{Are requested cores<br/>> available cores?}
    HardwareCheck -->|Yes| CapCores[Cap nProc to<br/>available hardware]
    HardwareCheck -->|No| Execution
    CapCores --> Execution
    
    %% Execution
    Execution[Run FDS solver]

    Execution --> PostProcess[Convert CSV to XLSX]
    
       
    PostProcess --> End([End Script])

    %% Styling
    style ExitError fill:#f00,stroke:#333
    style MeshDecomp  fill:#ff0,stroke:#333
    style RestartLogic fill:#0f0,stroke:#333
    style HardwareCheck fill:#0ff,stroke:#333
    style Execution fill:#4a0,stroke:#333
    style End fill:#f00,stroke:#333
```

## OpenFOAM

```mermaid
graph TD
    Start([Start Script]) --> EnvSetup[<b>1. Environment Setup</b><br/>Determine Software & Version<br/>Source OpenFOAM bashrc]
    
    EnvSetup --> Hardware[<b>2. Hardware Detection</b><br/>Calculate Physical Cores<br/>Remove legacy folders]
    
    Hardware --> Compilation{Custom Code to compile?}
    Compilation -- Allwmake/wmake --> Compile[Compile User Solver]
    Compilation -- No --> SanityCheck[<b>3. Sanity Checks</b><br/>Validate controlDict location<br/>Generate .foam files]
    
    Compile --> SanityCheck
    
    SanityCheck --> Decompose[<b>4. Parallel Config</b><br/>Set numberOfSubdomains<br/>Calculate Hierarchical Coeffs]
    
    Decompose --> Allrun{Is Allrun file present?}
    Allrun -- Yes --> ExecAllrun[Execute Allrun]
    Allrun -- No --> MeshCheck{Mesh Exists?}

    MeshCheck -- No --> MeshGen[<b>5. Mesh Generation</b>]
    MeshGen --> BlockMesh[blockMesh]
    BlockMesh --> SurfaceFeature[surfaceFeatures<br/>surfaceFeatureExtract]
    SurfaceFeature --> Snappy[decomposePar<br/>snappyHexMesh<br/>reconstructParMesh]
    Snappy --> MeshVerification[renumberMesh<br/>checkMesh<br/>createPatch<br/>changeDictionary<br/>scaleFactor]
    
    MeshCheck -- Yes --> SolverInit[<b>6. Solver Initialization</b><br/>Copy 0.org/0.orig to 0<br/>Set controlDict/stopAt endTime<br/>Set controlDict/startFrom latestTime]

    MeshVerification --> SolverInit
    SolverInit --> SolverDecompose[decomposePar]
    SolverDecompose --> Potential{controlDict/potentialFoam == true}
    
    Potential -- Yes --> PotExec[Execute potentialFoam]
    Potential -- No --> SolveLoop

    PotExec --> SolveLoop[<b>7. Solver Run</b><br/>solver specified in controlDict/application]
    
    SolveLoop --> Recon[<b>9. Reconstruction</b><br/>reconstructPar<br/>Delete processor folders]
    
    ExecAllrun --> End([End Script])
    Recon --> End([End Script])

    %% Styling
    style EnvSetup fill:#f9f,stroke:#333
    style Compilation fill:#99f,stroke:#333
    style MeshCheck fill:#0f0,stroke:#333
    style Allrun fill:#f00,stroke:#333
    style Potential fill:#ff8,stroke:#333
    style SolveLoop fill:#bfb,stroke:#333
    style Recon fill:#fbb,stroke:#333
```

## CUSTOM

```mermaid
graph TD
    Start((Start)) --> FindShell{Find *.sh files}
    
    %% Shell Script Loop
    FindShell -- Files Found --> ExecShell[Set Permissions: chmod ugo+x]
    ExecShell --> RunShell[Run bash script]
    RunShell --> LogShell[Save output to .log]
    LogShell --> FindShell
    
    %% Python Script Loop
    FindShell -- No more .sh --> FindPy{Find *.py files}
    FindPy -- Files Found --> Venv[Create venv: py-cloudhpc]
    Venv --> Activate[Activate venv]
    Activate --> Install[pip install -r requirements.txt]
    Install --> RunPy[Run python3 script]
    RunPy --> LogPy[Save output to .log]
    LogPy --> FindPy
    
    FindPy -- No more .py --> End((End))

    style Start fill:#f9f,stroke:#333
    style FindShell fill:#ff0,stroke:#333
    style FindPy  fill:#ff0,stroke:#333
    style End fill:#f9f,stroke:#333
    style RunShell fill:#bbf,stroke:#333
    style RunPy fill:#bbf,stroke:#333
```
